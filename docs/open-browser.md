# open-browser

`open-browser` åpner en statisk bildebrowser som allerede er laget med
[`make-browser`](make-browser.md).

Denne kommandoen blir trolig fjernet i en senere versjon, fordi HTML-filen kan
åpnes direkte fra PowerShell eller Filutforsker.

Eksempel fra PowerShell:

```powershell
.\index.html
```

Du kan også bruke TAB-fullføring:

```powershell
.\i<TAB>
```
