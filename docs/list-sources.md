# list-sources

`list-sources` viser kildene som er registrert i bildesamlingen.

## Referanse

```powershell
bildebank list-sources
```

## Hva kommandoen gjør

Kommandoen viser kilde-id, status, importtid, eventuelt hvilken kilde den er
erstattet av, navn og path. Den er nyttig når du vil kontrollere hvilke
importer Bildebank kjenner til.
