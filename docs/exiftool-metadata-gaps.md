# exiftool-metadata-gaps

`exiftool-metadata-gaps` finner metadata-datoer som ExifTool ser, men som
Bildebank ikke leser ennå.

## Referanse

```powershell
bildebank exiftool-metadata-gaps
bildebank exiftool-metadata-gaps --exiftool "C:\path\to\exiftool.exe"
```

Dette er en avansert feilsøkingskommando for å forbedre metadata-lesingen i
Bildebank.
