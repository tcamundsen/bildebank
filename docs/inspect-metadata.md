# inspect-metadata
<!-- CLI-HELP-START -->
```text
usage: bildebank inspect-metadata [valg] fil

positional arguments:
  fil         Bilde- eller videofil

options:
  -h, --help  show this help message and exit
```
<!-- CLI-HELP-END -->

`inspect-metadata` viser metadatafragmenter og datokandidater for en fil.

Eksempel:

```powershell
bildebank inspect-metadata 2026\01\image_001.jpg
```

Dette er en feilsøkingskommando når du vil se hva Bildebank klarer å lese ut av
en bilde- eller videofil.
