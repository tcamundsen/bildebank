# face-person-delete

`face-person-delete` sletter en person fra ansiktsdatabasen.

## Referanse

```powershell
bildebank face-person-delete "Krai"
```

Kommandoen krever bekreftelse. Den sletter personen, bekreftede
ansiktskoblinger og forslag for personen. Den sletter ingen bilder og ingen
scannede ansikter.
