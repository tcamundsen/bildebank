# errors

`errors` viser registrerte feil.

## Referanse

```powershell
bildebank errors
bildebank errors --limit 100
bildebank errors --stage scan
bildebank errors --include-resolved
```

## Hva kommandoen gjør

Kommandoen viser feil som Bildebank har lagret i databasen. Bruk den etter import
eller ved feilsøking.

`--include-resolved` viser også feil som senere er løst.
