# make-conflict-browser

`make-conflict-browser` lager en HTML-side for navnekollisjoner.

## Referanse

```powershell
bildebank make-conflict-browser
bildebank make-conflict-browser -o "konflikter.html"
```

Standardfilen heter `name-conflicts.html` og legges i bildesamlingen.

Bildene følger visningsrotasjonen som er lagret i Bildebank. Bildefilene
endres ikke.
