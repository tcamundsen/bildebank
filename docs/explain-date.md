# explain-date
<!-- CLI-HELP-START -->
```text
usage: bildebank explain-date [valg] fil

positional arguments:
  fil         Bilde- eller videofil

options:
  -h, --help  show this help message and exit
```
<!-- CLI-HELP-END -->

`explain-date` forklarer hvilken dato Bildebank ville brukt for en fil.

Eksempel:

```powershell
bildebank explain-date "C:\Bilder\IMG_1234.jpg"
```

Kommandoen viser aktuelle datokandidater og hvilken dato Bildebank ville valgt.
