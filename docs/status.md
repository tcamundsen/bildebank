# status

`status` viser en kort oppsummering av bildesamlingen.

## Referanse

```powershell
bildebank status
```

## Hva kommandoen gjør

Kommandoen teller importerte bilder og videoer. Viser også hvor mange filer
som har dato fra henholdsvis metadata, filnavn og mtime (når filen sist
ble endret).
