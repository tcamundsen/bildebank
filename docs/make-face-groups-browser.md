# make-face-groups-browser

`make-face-groups-browser` lager en HTML-side for ansiktsgrupper.

## Referanse

```powershell
bildebank make-face-groups-browser
bildebank make-face-groups-browser -o "face-groups.html"
```

Standardfilen heter `face-groups.html`.

Siden viser én gruppe om gangen. Bruk pil venstre og pil høyre for å bla mellom
gruppene. Klikk på et bilde for å åpne det i full størrelse.
